# Limited-media project world v0.1

## Purpose and hierarchy

Limited-media pages provide truthful understanding when media is absent, private, unavailable, documentary, chronology-led, or intentionally text-led. The order emphasizes narrative, evidence, limitations, process, and provenance before optional releases, links, relations, and context navigation.

The quiet shell and document-led layout use a reading measure, balanced metadata/evidence density, and a constrained type-led stage. Absence is treated as record state, not a visual defect.

## Media, proof, and fallbacks

The world never invents a thumbnail, diagram, screenshot, hero, skeleton, or repeated apology. Private media and source availability are described without URLs or local paths. Limitations and provenance receive stronger visual seams while remaining normal document sections.

The composition is complete with no JavaScript and at 320px, zoom, forced colors, reduced motion, and print. Authored marks are Level 0. Limited-media no-media, minimum-valid, private evidence, chronology, and long-title fixtures are required approval checks.
