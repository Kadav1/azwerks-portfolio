# Project world media policy v0.1

World media policy may choose stage measure, alignment, neutral surround, sequence rhythm, caption measure, lazy-loading posture, and lead preference. It cannot bypass publication, availability, rights, source safety, alt text, intrinsic dimensions, transcripts, or privacy.

All worlds use `crop: never`. Art uses the neutral art surround and `object-fit: contain`; portrait, landscape, and square audit fixtures remain uncropped. Software prefers real interface evidence. Visual system prefers explanatory specimens. Technical system prefers accessible diagrams. Limited media prefers a document or type-led state and never invents an image.

Audio and video retain native controls and visible transcript text. Video does not autoplay. Documents remain descriptive native links. No remote viewer, tracking iframe, carousel, lightbox, image SDK, nonexistent responsive variant, or upscaling rule is added.

Missing media remains a valid state. The existing no-media component describes availability once, without broken frames, apology, loading skeletons, or fabricated screenshots.
